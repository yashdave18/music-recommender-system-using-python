This is a content-based music recommender system built using Python and Streamlit. The system uses a Spotify dataset combining high and low popularity songs to recommend music similar to any selected track. It works by extracting features like artist, genre, and subgenre from each song and combining them into a single tags column. These tags are then vectorized using CountVectorizer and cosine similarity is computed between all songs to find the most similar ones.

The project is built with Python, Streamlit, Scikit-learn, Pandas, and NumPy. To run it locally, clone the repository, install dependencies, and launch the app with `streamlit run app.py`. The dataset was sourced from Kaggle's Spotify Music Dataset and the similarity matrix is precomputed and stored as a pickle file for fast recommendations.
